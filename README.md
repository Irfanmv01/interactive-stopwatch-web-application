# Stopwatch Web Application

A simple stopwatch web application developed using HTML, CSS, and JavaScript.

## Description
This application allows users to measure elapsed time using start, stop, and reset controls. The stopwatch updates the time in real-time using JavaScript.

## Features
- Start the stopwatch
- Stop the stopwatch
- Reset the stopwatch
- Displays time in hours, minutes, and seconds

## Technologies Used
- HTML
- CSS
- JavaScript

## Working Principle
- The application uses the JavaScript `setInterval()` function to update time every second.
- DOM manipulation is used to update the displayed time.
- Event listeners handle user interactions.
